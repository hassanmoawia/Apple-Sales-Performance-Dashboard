# Apple-Sales-Performance-Dashboard

📊 Project Overview

This Power BI project provides a comprehensive analysis of global sales operations, focusing on revenue drivers, customer segmentation, and regional performance. 

🚀 Key Features

Multi-Dimensional Revenue Analysis: Developed a dashboard tracking $18.04M in total revenue and 23K units sold, identifying Mac and iPhone as the primary sales drivers. 

Geographic & Segment Insights: Visualized global sales across North America, Europe, Asia, and Australia, revealing that Individual and Business segments each contribute over $4.4M in revenue. 

Time-Series Performance: Tracked monthly fluctuations to identify seasonal peaks, specifically highlighting a high-growth period in October with revenues reaching approximately $1.65M. 

🛠️ Tech Stack

Tool: Power BI Desktop 

Data Source: Apple Sales Records (Global) 

Key Skills: Data Modeling, DAX, Interactive Visualization
